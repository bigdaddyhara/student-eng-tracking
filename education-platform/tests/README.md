# Test Layout Foundation

## Scope
Test structure baseline aligned to CognitivePulse architecture.

## Areas
- tests/frontend/student-client: student client unit and integration tests.
- tests/frontend/teacher-dashboard: teacher dashboard unit and integration tests.
- tests/backend/realtime: real-time ingestion and transport tests.
- tests/backend/intelligence-fusion: pulse fusion and suggestion engine tests.
- tests/backend/persistence-history: storage and historical retrieval tests.
- tests/integration/classroom-pulse: classroom-level signal flow validation.
- tests/integration/end-to-end: full workflow validations across clients and services.
